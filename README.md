## 🚀 Overview

This project uses deep learning to detect **Chicken Pox symptoms from skin images**, enabling automated medical analysis.

---

## ✨ Key Features

- 🧠 CNN-based image classification  
- 🧹 Data preprocessing & augmentation  
- 📊 High accuracy classification  
- 🔍 Binary classification (infected vs non-infected)  

---

## 🧠 Tech Stack

- Python  
- TensorFlow / Keras  
- OpenCV
